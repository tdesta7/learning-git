# learning-git

This repo was used to learn github for biggner to use git.


…or create a new repository on the command line
echo "# git-learning" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:tdesta7/git-learning.git
git push -u origin main


…or push an existing repository from the command line
git remote add origin git@github.com:tdesta7/git-learning.git
git branch -M main
git push -u origin main
